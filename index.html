携帯端末制限と個人情報保護について

本アプリは許可された携帯端末（スマートフォン）でのみ稼働するセキュリティ対策版です。読み取った名刺データはブラウザ内の処理のみで完結し、外部のサーバーには保存されないため安全です。

🔒 携帯端末セキュリティ制限について

本アプリは社外利用や不正アクセスを防止するため、以下の二重の制限を設けています。

  1. デバイス制限（PCアクセス遮断）: PCからのアクセスは自動的にロック画面となり、利用できません。スマートフォンからアクセスしてください。
  2. セキュリティ承認（初回のみ）: 初回起動時に「アクセスコード」の入力が必要です。認証後は再入力は不要です。
  3. 端末固有ID (UUID): 各端末には自動で固有ID（UUID）が割り当てられます。管理者が「どの端末でアプリが認証されたか」を照合する際に使用されます。

📱 ホーム画面への追加（全画面表示）

スマートフォン（iPhone / Android）のホーム画面に追加することで、アドレスバーが非表示になり、通常のスマホアプリと同様に全画面で起動できるようになります。

🍎 iPhone (Safari) の設定手順

  1. Safari でアプリのURLを開きます。
  2. 画面下部中央にある「共有」ボタンをタップします。
  3. メニューを下へスクロールし、「ホーム画面に追加」を選択します。
  4. 画面右上の「追加」をタップします。

🤖 Android (Chrome) の設定手順

  1. Chrome でアプリのURLを開きます。
  2. 画面右上にある「3つのドット（メニュー）」アイコンをタップします。
  3. メニューから「アプリをインストール」（または「ホーム画面に追加」）を選択します。

📖 基本的な使い方

  1. ホーム画面のアイコンからアプリを起動します。
  2. 初回起動時は、アクセスコードを入力して承認を行います。
  3. 右上の歯車マーク（設定）から「Gemini APIキー」が正しく設定されていることを確認します。（自動引き継ぎがある場合は不要）
  4. 「カメラを起動して名刺を撮影」または「ファイルから写真を選択」から名刺画像をセットします。
  5. 「名刺をスキャンする」ボタンをタップします。
  6. 解析されたテキストを「コピー」し、社内システムやExcel等に貼り付けて活用してください。

⚙️ 管理者向け情報

  • 端末の再ロック（認証解除）: 社員が端末を返却・紛失した際など、認証をクリアしたい場合は、設定画面（右上の歯車）の下部にある「この端末を再ロックする」を実行してください。
  • 端末ID（UUID）のコピー: 初回起動画面および設定画面で、その端末のUUIDを確認・コピーできます。
