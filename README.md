# AI名刺読み取りアシスタント（端末制限版）- 導入・管理手順書

本アプリは、スマートフォンのカメラで名刺を撮影するだけで、AI（Gemini）が会社名、氏名、連絡先などを自動的に整理・抽出してくれる社内専用の便利ツールです。

このバージョンでは、社内セキュリティ保護と個人情報の取り扱い制限のため、**「PCからのアクセス制限」**および**「許可された携帯端末（スマートフォン）のみの利用制限」**の二重の携帯端末制限機能が実装されています。

---

## 🔒 携帯端末制限（セキュリティ仕様）について

本アプリは、社外や許可されていない端末、PCからの不正な利用を防ぐため、以下のセキュリティ制限が設けられています。

### 1. デバイス制限（PCアクセスの遮断）
- ユーザーエージェント（UA）およびタッチスクリーンの有無を判定し、PCからのアクセスを自動的に遮断します。
- PCで開いた場合は、画面上に「PCアクセス制限」のロック画面が表示され、アプリを操作することができません。

### 2. パスコード認証（初回起動時）
- スマートフォンで初めてアプリを起動した際、**セキュリティ承認画面**が表示されます。
- 管理者から配布された**アクセスコード**を入力して「アクセスを承認する」をタップすることで、名刺読み取り機能が解放されます。
- 解除状態はブラウザ（Cookie/LocalStorage）に保存されるため、2回目以降の起動時はコードの再入力は不要です。
- **デフォルトのアクセスコード**: `MOBILE-ACCESS-2026`

### 3. 端末固有識別ID (UUID)
- 初回起動画面およびアプリ内の「設定」メニューに、この端末固有の「端末識別ID (UUID)」が表示されます。
- 各社員の社用携帯を管理者が把握し、「どの端末でアプリが認証されたか」を登録・照合する際に利用できます。

## 📱 スマートフォンへのインストール方法（ホーム画面に追加）

スマートフォン（iPhone / Android）のホーム画面に追加することで、アドレスバーが非表示になり、通常のスマホアプリと同様に全画面表示で快適に利用できるようになります。

### 🍎 iPhone (Safari) での設定手順
1. **Safari** を起動し、本アプリの公開URLを開きます。
2. 画面下部の中央にある **「共有」ボタン（□から上矢印が出ているアイコン）** をタップします。
3. 表示されたメニューを下にスクロールし、**「ホーム画面に追加」** をタップします。
4. 画面右上の **「追加」** をタップします。
5. ホーム画面に **専用アイコン（「名刺読取(制限)」）** が追加されます。

### 🤖 Android (Chrome) での設定手順
1. **Chrome** を起動し、本アプリの公開URLを開きます。
2. 画面右上にある **「3つのドット（メニュー）」** アイコンをタップします。
3. メニューの中から **「アプリをインストール」**（または「ホーム画面に追加」）をタップします。
4. 確認画面が表示されるので「インストール」または「追加」をタップします。

## 📖 基本的な使い方

1. ホーム画面のアイコンからアプリを起動します。
2. 初回起動時は、アクセスコード（`MOBILE-ACCESS-2026`）を入力して制限を解除します。
3. 右上の歯車マーク（設定）をタップし、「Gemini APIキー」を設定します。（同じブラウザで議事録アプリをご利用中の場合は自動で引き継がれます）
4. **「カメラを起動して名刺を撮影」** または **「ファイルから写真を選択」** をタップし、名刺の画像をセットします。
5. **「名刺をスキャンする」** をタップすると、数秒でAIが解析したテキストが画面に表示されます。
6. **「コピー」** ボタンをタップすると、結果がクリップボードに一括コピーされます。

## ⚙️ 管理者向け情報

- **アクセスコードの変更**:
  `app.js` の `const CORRECT_PASSCODE = 'MOBILE-ACCESS-2026';` の部分を書き換えることで、承認に必要なアクセスコードを変更できます。
- **端末の再ロック（認証のリセット）**:
  社員が端末を返却する際や、紛失時などに認証を無効化したい場合は、アプリ内設定（右上の歯車）の下部にある「この端末を再ロックする」ボタンを押すことで、認証情報をクリアし、再度パスコード入力が必要な状態に戻すことができます。
- **テスト用PC表示モード**:
  開発時などでPCからデバッグしたい場合は、ブラウザのデベロッパーツールで `localStorage.setItem('ignore_mobile_check', 'true')` を実行し、ページをリロードすることでPCからのモバイルチェックをスキップできます。
